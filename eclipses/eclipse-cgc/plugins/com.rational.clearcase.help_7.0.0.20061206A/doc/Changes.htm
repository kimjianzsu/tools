Changes from the previous major release

The following ClearCase features have changed or are no longer supported in the current version of the ClearCase SCM Adapter:

  • The Connect to Rational ClearCase option on the ClearCase menu can now be toggled on and off. When this option is on, the user is connected to ClearCase, a check mark appears next to the option name, and the ClearCase menu options are active. When this option is off, the user is disconnected from ClearCase, the check mark disappears, and the ClearCase menu options are no longer active.
  • The Window > Preferences > Workbench > Label Decorations > ClearCase preference is now the Window > Preferences > Workbench > Label Decorations > ClearCase SCM Adapter option
  • The Window > Preferences > Team > Rational ClearCase option is now the Window > Preferences > Team > Rational ClearCase SCM Adapter option
  • If the ClearCase Remote Client (CCRC) is also installed on your machine, your CCRC web views also appear in the ClearCase View dialog when you are importing a project set. If you select a CCRC web view to use for the import process, any future ClearCase operations you perform on the project set in Eclipse are accomplished through the ClearCase Remote Client.

The following option is new in the current version of the ClearCase SCM Adapter:

  • The ClearCase MVFS Support Team preference option

Support for the following platforms is new in the current version of the ClearCase SCM Adapter:

  • Red Hat Enterprise Linux 4.0
  • Sun Solaris
