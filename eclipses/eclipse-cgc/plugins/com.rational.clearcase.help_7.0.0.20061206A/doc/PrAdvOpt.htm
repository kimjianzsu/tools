Advanced Options

To open the Rational ClearCase Options dialog box, click Advanced Options in the ClearCase SCM Adapter preferences dialog. 

This option becomes active after you connect to ClearCase.

Use the Rational ClearCase Options dialog box to specify the following preferences:

General tab:
Search for new project files
Enable the Get Latest Version command on the Source Control menu

Operations tab:
Check Out
    Reserved
    Unreserved if already reserved
    Preserve file modified time on checkout

Check In
    Checkin even if identical
    Preserve file modified time on checkin

Undo Checkout
    Save copy of file with a '.keep' extension

Activity
    Always prompt for an activity when working in a view of a ClearCase project
