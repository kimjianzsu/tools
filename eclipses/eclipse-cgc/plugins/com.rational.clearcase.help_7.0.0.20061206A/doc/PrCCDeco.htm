ClearCase decorations

To specify the decoration of Rational Developer files under ClearCase control, use the ClearCase decorations preference option.

To view icons next to ClearCase elements, select Enable Icon Decorations. To view text indicators of the status of ClearCase elements, select Enable Text Decorations. Select Enable Text and Icon Decorations to view both icons and text indicators. 

Note: These options do not take effect unless the following preference is also set: Window > Preferences > Workbench > Label Decorations > ClearCase SCM Adapter.

By default, this option is set to Enable Icon Decorations.
