Importing a WebSphere Studio 4.x project

To import a WebSphere Studio 4.x project that is under Rational ClearCase source control:

  1. In the Rational Developer product, click File > Import.
  2. In the Import dialog box, click Existing WebSphere Studio 4.x ClearCase Project into Workspace.
  3. Follow the wizard instructions to import the WebSphere Studio 4.x project.

NOTE: When migrating a Java project, make sure the WebSphere Studio 4x project is under source control before importing it. You can place the project under source control by using clearfsimport instead of copy to move the project between VOBs or by adding the project to source control after copying. 

The project is now available and under source control.
