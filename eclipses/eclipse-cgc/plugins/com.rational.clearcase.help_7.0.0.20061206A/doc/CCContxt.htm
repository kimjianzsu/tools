Team context menu

To open the Team context menu in the Rational Developer product, right-click on a Rational Developer project that is under ClearCase source control and then click Team > Share Project. In the Share Project window, select ClearCase SCM Adapter and click Next. Complete the steps of the Share Project wizard. The following ClearCase items become available.

Item Name
Add to Source Control
Check Out
Check In
Undo Check Out
Update
Refresh Status

Undo Hijack
Compare with Previous Version
