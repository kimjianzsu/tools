Import an EAR file to the workspace

To import an existing EAR file into your workspace and place it under Rational ClearCase control:

  1. In the Rational Developer product, choose File > Import.
  2. In the Import dialog box, select EAR file and click Next.
  3. In the Import dialog box, specify the EAR file and the name of the EAR project, and click Finish.

To place imported EAR files under source control, you must move each EAR project to a ClearCase view.

To move an EAR project to a ClearCase view:

  1. Select the EAR project in the Rational Developer product and click Edit > Move. Specify a project location that is a folder beneath a VOB for which you have created a view.
  2. Verify that the appropriate files are selected in the Add Element(s) to Source Control dialog box, and click OK.
  3. If you are prompted to specify an activity, select an existing activity or create a new one.
  4. Repeat steps 4-6 for any other Web or EJB projects that are associated with the EAR.
  5. In the Rational Developer product, right-click the EAR project and choose Properties. 
  6. In the Properties dialog box, select Project References, clear the projects you moved, and select their new equivalents. Click OK.
