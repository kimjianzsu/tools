Check In

To create a new version of a file or folder element that you previously checked out, right-click the element in Rational Developer product and click Check In on the Team context menu or click the ClearCase Check In icon.

This option becomes active after you connect to ClearCase and select a ClearCase element in the Rational Developer product that is checked out to your view.
