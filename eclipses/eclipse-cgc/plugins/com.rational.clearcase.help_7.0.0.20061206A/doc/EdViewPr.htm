Edit View Properties

To open the properties page for the current view, choose ClearCase > Edit View Properties in the Rational Developer product. In the Edit View Properties dialog box, select a view and click OK. 

When the Properties dialog box opens, you can do the following:

  • Read information about the view
  • Read and edit the config spec for the view (if it is not a ClearCase UCM view)
  • Change the view's association with a view profile
  • Edit the load rules of a snapshot view

This option becomes active after you connect to ClearCase.
