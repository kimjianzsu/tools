Build command

This ClearCase SCM Adapter preference specifies the build command that ClearCase runs when you click ClearCase > Build. To specify the build command, select a default omake or clearmake build command in the Build command box or type your own command. The default build commands search for a makefile in the project workspace directory that has the same name as the Rational Developer project and a .mak file name extension. 

Note: This option does not apply to ClearCase LT.

By default, this option is set to omake -s -f "${ProjPath}${ProjName}.mak"
