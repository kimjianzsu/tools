Importing an Eclipse Team Project Set

To import an Eclipse Team Project Set, take the following steps

  1. In Eclipse, click File > Import.
  2. If you are using Eclipse 3.2x, click Team > Team Project Set. 

    If you are using Eclipse 3.1x or 3.0x, click Team Project Set.
  3. Select the project set file (.psf) of the project you want to import and click Finish.

    If the project set file is in a ClearCase VOB, the import process is complete and you can access the project through the ClearCase view you were using when you performed the import. If the project set file is not in a ClearCase VOB, proceed to steps 4 through 7.
  4. The ClearCase View dialog appears. If you are not connected to ClearCase, click the Connect to ClearCase button.
  5. The views that are currently running on your system appear in the ClearCase View dialog. If you want to select a dynamic view that is not currently running on your system, click the Start a different Dynamic view button and select the view to start in the Start View dialog.
  6. Select the view you want to use to import the project set in the ClearCase View dialog.

    NOTE: If the ClearCase Remote Client (CCRC) is also installed on your machine, your CCRC web views also appear in the ClearCase View dialog. If you select a CCRC web view, any future ClearCase operations you perform on the project set in Eclipse are accomplished through the ClearCase Remote Client.
  7. Click OK to import the project set using the ClearCase view you selected
