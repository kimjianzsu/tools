Run Merge Manager

To combine your changes on one ClearCase branch with changes on another branch, click ClearCase > Run Merge Manager in the Rational Developer product. 

The view that you want to merge to is the destination view. Perform the merge in the destination view.

This option becomes active after you connect to ClearCase.
