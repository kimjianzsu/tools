Enabling the ClearCase SCM Adapter capability

If you are using the ClearCase SCM Adapter with Eclipse, the ClearCase SCM Adapter capability is enabled by default. 

If you are running the ClearCase SCM Adapter with any of the Rational Developer products, the ClearCase SCM Adapter capability is off by default and you must enable this capability before you can use ClearCase options.

To enable the ClearCase SCM Adapter capability:

Select the Window > Preferences > Workbench > Capabilities > Team > ClearCase SCM Adapter option in the Rational Developer product. After activating the ClearCase SCM Adapter, the ClearCase menu options and icons appear in Rational Developer products.

Some operations can automatically activate the ClearCase SCM Adapter such as, for example, importing a Rational Developer project or project set from a VOB or sharing a Rational Developer project.

Note: If you have enabled the ClearCase SCM Adapter capability but the ClearCase menus and icons do not appear in your perspective, you can use the following method to manually add the ClearCase SCM Adapter to the perspective:

  1. Click the Window > Customize Perspective option.
  2. Click the Commands tab in the Customize Perspective window.
  3. Select the ClearCase SCM Adapter option in the Available command groups list box.

To begin to use the ClearCase SCM Adapter, you must connect to ClearCase.
