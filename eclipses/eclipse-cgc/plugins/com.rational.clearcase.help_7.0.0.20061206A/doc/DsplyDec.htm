Display ClearCase decorations

To display the ClearCase text or icon decorations:

  1. In the Rational Software Development Platform, click Windows > Preferences.

  2. In the Preferences dialog box, expand the Workbench entry and select Label Decorations.

  3. In the Available Label Decorations box, select ClearCase SCM Adapter. 

  4. In the Preferences dialog box, expand the Team entry and select ClearCase SCM Adapter.

  5. Select an option in the ClearCase Decorations box and click OK or Apply. 

By default, icon decorations are enabled.
