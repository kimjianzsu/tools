When workspace is closed

This ClearCase SCM Adapter preference specifies Rational ClearCase behavior when closing a Rational Developer workspace.

If you want to see a list of modified elements and choose the elements to check in, select Prompt to checkin. If you want to check in elements without being notified, select Automatically checkin. If you do not want either option, select Do nothing.

By default, this option is set to Prompt to checkin.
