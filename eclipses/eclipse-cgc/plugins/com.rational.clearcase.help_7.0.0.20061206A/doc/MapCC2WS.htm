Mapping ClearCase concepts to Rational Developer products

To store Rational Software Development Platform projects in ClearCase UCM components, create a ClearCase view and a stream. The stream supplies the configuration to the view which determines the correct versions to select from the components.

You can import Rational Software Development Platform projects from a ClearCase view into a Rational Software Development Platform workspace.
