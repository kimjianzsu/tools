Refresh Status

To update the state of the currently selected ClearCase element in the Rational Developer product, click Refresh Status on the Team context menu or click the ClearCase Refresh Status icon. If you have performed ClearCase operations (such as, for example, a checkin or checkout operation) on the element outside the Rational Developer product, performing this operation updates the ClearCase status of the element in the Rational Developer product.

This option becomes active after you click ClearCase > Connect to Rational ClearCase and select a ClearCase element in the Rational Developer product.
