Request status information on demand only

If this ClearCase SCM Adapter preference is selected, the status of only those elements that you are currently viewing is updated in the Rational Developer product, rather that updating the status of all the elements in your project. If fewer than fifty files or folders are not under source control, you are prompted to add them. If more than fifty files or folders are not under source control, click Find Resources to Add to Source Control to identify them.

The accelerator keys for this option are Alt + Q. 

This option is on by default.
