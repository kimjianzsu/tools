ClearCase text decorations

The following text decorations are appended to the names of elements in Rational Developer products to denote their status in Rational ClearCase:

Text Description

@@

Indicates that the element is under ClearCase source control.

@@CO

Indicates that the element is currently checked out of ClearCase.

@@HJ

Indicates that the element is currently hijacked. 

@@??

Indicates that source control status has not been determined.
