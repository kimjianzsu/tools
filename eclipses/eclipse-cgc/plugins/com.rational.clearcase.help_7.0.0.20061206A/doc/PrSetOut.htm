Set default to check out files after adding them to source control

This ClearCase SCM Adapter preference specifies the default setting of the Keep checked out option in the Add Element(s) to Source Control dialog box.

To set the Keep checked out option in the Add Element(s) to Source Control dialog box by default, select this option. To set the Keep checked out option default state to off, clear this check box.

This option is only available on Windows platforms and is off by default.
