Check Out

To modify a Rational Developer file or folder that is under Rational ClearCase source control, right-click the element in the Rational Developer product and click Check Out on the Team context menu or click the ClearCase Check Out icon. You can then edit file elements, or add and delete file elements and symbolic links in directory elements.

This option becomes active after you connect to ClearCase and select an element in the Rational Developer product that is not checked out to your view.
