Working in disconnected mode

To work in disconnected mode, take the following steps:

  1. Update your snapshot view.
  2. Disconnect from ClearCase by clicking ClearCase > Connect to ClearCase to toggle it off.
  3. Disconnect from the network.
  4. Start the Rational Developer product but do not connect to ClearCase. You may need to clear the Automatically Connect to ClearCase at Startup preference check box.
  5. Use the Hijack option to modify files. Using this option makes read-only files in a snapshot view writeable. 

    Avoid operations that rename or move files as these are difficult to reconcile after reconnecting.
  6. Reconnect to the network, connect to ClearCase (note that it may take several minutes before you can perform ClearCase operations), update the view by clicking ClearCase > Update View, resolve any hijacked files, and use the Find Resources to Add to Source Control option.
