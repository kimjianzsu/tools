Associate a workspace with a ClearCase view 

It is a good practice to maintain a separate Rational Developer workspace for each Rational ClearCase view that you use.

To associate a workspace with a ClearCase view:

  1. To create a new view, click ClearCase > Create New View.
  2. Create a new workspace storage directory that corresponds to the ClearCase view: "workspace_view_tag". 

When you start the Rational Developer product, you select a workspace in the Workspace Launcher. 

To move between workspaces, click File > Switch Workspace in the Rational Developer product.
