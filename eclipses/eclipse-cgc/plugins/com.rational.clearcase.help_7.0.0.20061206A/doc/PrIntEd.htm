When checked-in files are edited by an internal, interactive editor

This ClearCase SCM Adapter preference specifies Rational ClearCase behavior when you begin to interactively edit a file in the Rational Developer product that is not checked out. 

If you want to be prompted to check out the file before editing it, select Prompt to checkout. If you want to check the file out without being notified, select Automatically checkout. If you do not want either option, select Do nothing.

By default, this option is set to Prompt to checkout.
