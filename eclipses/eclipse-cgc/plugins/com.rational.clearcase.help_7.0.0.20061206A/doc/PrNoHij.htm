Do not hijack in snapshot views 

If this ClearCase SCM Adapter preference is set, Rational ClearCase does not automatically hijack files when any of the following ClearCase SCM Adapter preference options are set to Automatically checkout:

  • Checked in files are edited by an internal, interactive editor
  • Checked in files are edited by an internal, non-interactive editor
  • Checked in files are saved by an internal editor

By default, this option is on.
