Migrate a Project in ClearCase to the Rational Software Development Platform

To migrate an existing project that was not created in the Rational Software Development Platform but which is under ClearCase source control, take the following steps.

  1. In WebSphere Studio, choose File > New > Project. The New Project dialog box appears.
  2. Within the New Project dialog box, select a project type and click Next. The New dialog box appears.
  1. Within the New dialog box, specify a project name, uncheck the Use default option, and enter a directory path or use the Browse button to select the path. The project information you specify (such as the project name, location, type, and builders) must match the existing project.

Note: If the project is an Enterprise Java Beans (EJB) project, refer to the following IBM documentation for information migrating EJB projects to external version control systems:

  • Choose Start > Programs > Rational Application Developer > Readme to open the Readme file, then click on the Installation and migration guide link.
  • Within VisualAge for Java 4.0, choose Help > Concepts > Interface to external version control systems.
