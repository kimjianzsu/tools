Perform Refresh Status operations recursively

This Rational ClearCase preference option specifies Rational ClearCase behavior when you perform a Refresh Status operation.

If this preference is set, selecting a Rational Developer folder or project and performing a Refresh Status operation updates the status of all files or folders in or below the currently selected folder or project. If this preference is not selected, a Refresh Status operation updates only the currently selected file, folder, or project.

This option is on by default.
