Rebase Stream

To start or continue a Rational ClearCase rebase operation, which changes your development stream to include the versions in a recommended baseline, click ClearCase > Rebase Stream or click the ClearCase Rebase Stream icon. Your development stream continues to include versions associated with any undelivered activities.

This option becomes active after you select the ClearCase > Connect to Rational ClearCase menu option.
