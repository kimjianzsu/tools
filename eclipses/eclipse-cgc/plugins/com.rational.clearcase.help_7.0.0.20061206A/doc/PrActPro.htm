Always prompt for an activity when working in a view of a ClearCase project

This Advanced Options preference specifies Rational ClearCase UCM behavior when you perform checkin or checkout operations. When this option is set, you are prompted to select an activity after you click OK in the dialog box that you are using. When this option is not selected, the operation is associated with the view's current activity.

This option is on by default.
