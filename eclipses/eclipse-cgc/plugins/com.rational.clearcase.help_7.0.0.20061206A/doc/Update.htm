Update

To update the currently selected elements in a snapshot view, right-click the elements in the Rational Developer product and click Update on the Team context menu or click the Update icon. The version of the element that ClearCase copies into the view depends on the configuration specification that is set for that view.

This option becomes active after you connect to ClearCase and select a ClearCase element within the Rational Developer product.
