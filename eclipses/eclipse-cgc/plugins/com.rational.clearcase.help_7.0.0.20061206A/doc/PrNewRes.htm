When new resources are added

This ClearCase SCM Adapter preference specifies Rational ClearCase behavior when it detects new Rational Developer product resources. A new Rational Developer product resource is any file that is generated by Rational Developer products.

If you want to be prompted to add any new resources to source control, select Prompt to add to source control. If you want to add any new resources to source control, select Automatically add to source control. If you do not want either option, select Do nothing.

By default, this option is set to Prompt to add to source control.
