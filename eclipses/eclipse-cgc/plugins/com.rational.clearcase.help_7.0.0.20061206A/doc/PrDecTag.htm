Decorate project names with view tags

This ClearCase SCM Adapter preference controls the display of the ClearCase view tag after the name of the Rational Developer project root directory. 

If you select this option, the ClearCase view tag appears in brackets after name of the ClearCase Developer project root directory. For example, if this name is my_proj and your view tag is my_view, the project root directory name appears in the Rational Developer product as: 

my_proj [my_view]

If you do not want to see the view tag, clear this check box.

By default, this option is on.
