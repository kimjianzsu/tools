Reserved

This Advanced Options preference specifies Rational ClearCase behavior when checking out elements. When this preference is selected, other users cannot check out the same element from the same branch at the same time. By default, ClearCase attempts to perform a reserved checkout when you check out a ClearCase file or folder element. 

This option is on by default.
