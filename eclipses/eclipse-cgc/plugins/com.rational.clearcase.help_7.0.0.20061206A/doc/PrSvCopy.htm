Save copy of file with a '.keep' extension

This Advanced Options preference specifies Rational ClearCase behavior when you cancel a checkout. When this preference is selected, a copy of each file element that was involved in the Undo Checkout operation is saved and a .keep extension is appended to each file name. If this option is not selected, the files that were involved in the undo checkout operation are deleted and any edits that you may have made to those files are lost.

This option is only available on Windows platforms and is on by default.
