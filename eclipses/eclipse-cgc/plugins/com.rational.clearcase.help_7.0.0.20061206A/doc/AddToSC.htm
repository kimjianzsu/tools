Add to Source Control

To place Rational Developer files or folders under Rational ClearCase control, right-click the files or folders and click Add to Source Control from the Team context menu or click the Add to Source Control icon. Use the Add Element(s) to Source Control dialog box to place the selected files or folders under ClearCase source control. 

Select the Keep checked out option in the Add Element(s) to Source Control dialog box if you want the elements to be checked out after ClearCase adds them to source control. 

Note: You can sort the files or folders listed in the Add Element(s) to Source Control dialog box by clicking on the column headings.

This option becomes active after you connect to ClearCase and then select a Rational Software Development Platform file or folder that is not under ClearCase source control.
