ClearCase icon decorations

Icon decorations appear next to elements in the Rational Developer product to denote their status in ClearCase:

 A blue underlay indicates that the element is under source control.

A checkmark indicates that the element is currently checked out.

A question mark overlay indicates that source control status has not been determined. 

The Hijack icon decoration indicates that the element is currently hijacked.
