Unreserved if already reserved

This Advanced Options preference specifies Rational ClearCase behavior when you attempt to check out files or folders that are checked out to another view as reserved. When this option is selected, your checkout is unreserved.

This option is active when the Check Out Reserved preference is selected in the Advanced Options dialog box.

This option is off by default.
