Supported software and specifications

The ClearCase SCM adapter requires the following software:

  • One of the following platforms: Windows 2000, Windows XP Professional, SLES 9, Red Hat Enterprise Linux 2.1, Red Hat Enterprise Linux 3.0, Red Hat Enterprise Linux 4.0, or Sun Solaris
  • Rational Software Development Platform Version 6.x or 7.x
  • ClearCase 6.0 or later (ClearCase or ClearCase LT) with the latest patches 
    or 
    ClearCase 7.0.0 or later with the latest Fix Packs. 

    The following table lists the patches for each version of Rational ClearCase 2003.06.00:
