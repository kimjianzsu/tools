Show Version Tree

To start the Rational ClearCase Version Tree Browser, right-click an element in the Rational Developer product and select the Show Version Tree option from the Team context menu or click the ClearCase Show Version Tree icon. 

The Version Tree Browser provides a graphical display of the versions of an element, including any branches or UCM streams. The browser can show:

  • The number of versions, streams, and branches for an element
  • The relationship between the versions, streams, and branches
  • Metadata used to describe the element

You can also perform ClearCase operations on elements, branches, or streams in the Version Tree Browser.

This option becomes active after you connect to ClearCase.
