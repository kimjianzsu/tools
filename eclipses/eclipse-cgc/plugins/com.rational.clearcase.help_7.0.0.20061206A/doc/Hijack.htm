Hijack

To edit elements under source control offline, use the Hijack menu item. In the Rational Developer product, right-click the elements that you want to edit and click Hijack on the Team context menu.

This option applies only to snapshot views.

This option is active when you select a ClearCase element in the Rational Developer product.
