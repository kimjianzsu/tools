Find Checkouts

To specify the criteria that ClearCase uses to locate checked-out elements, click ClearCase > Find Checkouts. In the Find Criteria dialog box, specify the search location. By default, ClearCase searches the specified folder all its subfolders for any elements that you have checked out to the current view.

This option becomes active after you connect to ClearCase.
