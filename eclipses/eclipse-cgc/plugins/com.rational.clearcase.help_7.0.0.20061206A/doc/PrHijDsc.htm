Hijack in snapshot views when disconnected

If this ClearCase SCM Adapter preference is set and you are using a snapshot view while you are disconnected from ClearCase, Rational ClearCase automatically hijacks files instead of checking them out when this option is on and any of the following ClearCase SCM Adapter preference options are set to Automatically checkout:

  • Checked in files are edited by an internal, interactive editor
  • Checked in files are edited by an internal, non-interactive editor
  • Checked in files are saved by an internal editor

By default, this option is off.
