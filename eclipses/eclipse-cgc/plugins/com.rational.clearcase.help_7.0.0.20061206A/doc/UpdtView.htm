Update View

To start the Rational ClearCase Update Tool, which synchronizes the selected files and directories in a snapshot view with the contents of the VOB, click ClearCase > Update View or click the ClearCase Update View icon. The Update Tool may copy files or directories from the VOB, or remove files or directories from the view. This process ensures that the versions you modify in your view are the versions derived from the view configuration specification. 

This option becomes active after you connect to ClearCase and select a Rational Developer product file or folder.
