Create New View

To create a new view in the Rational Developer product, click ClearCase > Create New View. On the first page of the View Creation Wizard, take one of the following actions:

  • To work with source files for a ClearCase UCM project, click Yes. Then select your UCM project from the list.

    • On a ClearCase LT system, the list includes all projects that your ClearCase administrator makes available.

    • On a ClearCase system, the list includes all projects available in your network region.

  • To see information about a project in the list, select it and click Properties.
  • To work with source files for a project that does not use ClearCase UCM, click No.

Check with your project manager if you need to determine whether your project uses ClearCase UCM.

This option becomes active after you connect to ClearCase.
