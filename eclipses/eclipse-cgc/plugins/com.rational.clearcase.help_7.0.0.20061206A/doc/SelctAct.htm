Set Current Activity 

To associate a Rational ClearCase UCM activity with the current ClearCase view, use the ClearCase Set Current Activity option. Any future checkout of elements in this view is associated with the activity you select.

  1. Click ClearCase > Set Current Activity or click the ClearCase Set Current Activity icon.
  1. In the Set Activity ID dialog box, select a ClearCase activity and click OK.

You cannot create a new activity for the view during this operation. To create a new activity, you must check in or check out a file in this view.

This option becomes active after you connect to ClearCase.
