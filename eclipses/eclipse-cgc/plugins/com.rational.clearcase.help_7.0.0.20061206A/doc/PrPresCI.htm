Preserve file modified time on checkin

This Advanced Options preference specifies Rational ClearCase behavior when you check in file or folder elements. When this preference is selected, the time stamp of an element is not modified when you check it in. The time stamp continues to show the time that the element was last saved. Other programs do not detect a change to the element and prompt you unnecessarily. We recommend that you select this preference option.

This option is on by default.
