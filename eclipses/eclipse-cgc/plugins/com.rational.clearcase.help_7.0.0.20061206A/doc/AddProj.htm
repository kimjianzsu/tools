Import a Rational Developer project

You can import Rational Developer projects into your workspace and place them under Rational ClearCase control. 

To import a Rational Developer project into your workspace:

  1. In Rational Developer, click File > Import. 

  2. In the Import dialog box, select Existing Project into Workspace.

  3. Follow the instructions in the Import wizard.

You can also import a project set file (a .psf file) by clicking File > Import > Team Project Set in the Rational Developer product. If you import a project set file that is not in a VOB, ClearCase SCM Adapter prompts you to choose a view from which to load the projects in the .psf file.
