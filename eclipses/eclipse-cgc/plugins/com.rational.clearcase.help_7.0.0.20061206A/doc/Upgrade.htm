Upgrading the ClearCase SCM Adapter

To upgrade your version of the ClearCase SCM Adapter: 

  1. Select Help > Software Updates > Find and Install in the Rational Developer product. 
  2. In the Install dialog box, select a feature update method. 

    If you selected Search for updates of the currently installed features, click Next and follow the remaining steps in the Install dialog box.

    If you selected Search for new features to install, click Next and follow steps 3-6.   
  3. Select an Update Site and click Next.
  4. Select the features you want to install and click Next.
  5. Review the license agreements and, if you accept them, click Next.
  6. Select an install location and click Finish.
