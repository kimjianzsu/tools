Compare with Previous Version

To compare a version of a ClearCase element with the previous version, right-click the element in the Rational Developer product and click Compare with Previous Version on the Team context menu or click the Compare with Previous Version icon.

This option becomes active after you connect to ClearCase and select a Rational Developer file, folder, or project that is under ClearCase source control.
