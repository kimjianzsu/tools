Work Offline

To work offline with Rational Developer files that are under ClearCase control in a snapshot view:

  1. In the Rational Developer product, select the files that you want to edit, right-click, and click Hijack on the Team context menu.
  2. Edit and save the files offline.
  3. When you are ready to work online, click Connect to Rational ClearCase in the Rational Developer product. 
  4. Click ClearCase > Update View.
