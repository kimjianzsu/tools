Undo Hijack

To undo a Hijack operation, select the hijacked elements in the Rational Developer product, right-click, and click Undo Hijack on the Team context menu. If the operation is successful, the Hijack icon no longer appears next to these elements. 

This option applies only to snapshot views.

This option is active when you select hijacked elements in a Rational Developer product.
